mysql导入导出

发表于 2016-04-22   |   阅读次数

导出

例子：mysqldump -u root -p test>d:\test.sql
注意：此备份只备份数据和数据结构，没有备份存储过程和触发器

导入

mysql -u用户名 -p 数据库名 < 数据库名.sql
范例：
mysql -uabc_f -p abc < abc.sql
（导入数据库abc从abc.sql文件）

配置PHP7+laravel环境

发表于 2015-12-08   |   阅读次数

php7/php.ini

; extension=….
去掉;开启php扩展
extension=php_curl.dll
extension=php_mbstring.dll
extension=php_mysqli.dll
extension=php_openssl.dll
extension=php_pdo_mysql.dll

extension_dir = “PHP path”

apache/conf/httpd.conf

PHPIniDir “D:/phptools/php-7.0.0-Win32-VC14-x64/“
LoadModule php7_module “D:/phptools/php-7.0.0-Win32-VC14-x64/php7apache2_4.dll”

上面这句PHPIniDir不加报错Fatal error: Uncaught ReflectionException: Class log does not exist in D:\phpwww\laravel\vendor\laravel\framework\src\Illuminate\Container\Container.php:737 Stack

DocumentRoot “D:/phpwww/laravel/“
Directory “D:/phpwww/laravel/“

composer

需要使用openssl扩展所以需要Apache需要vc14
