Laravel Homestead安装记录

安装

vagrant
virtualbox
git
cd到镜像所在目录
vagrant box add laravel/homestead homestead.box
—-Usage: vagrant box add [options]
vagrant box list
git clone https://github.com/laravel/homestead.git Homestead
目录里运行init.sh，会在windows家目录下生成一个名为 .homestead的文件夹
—-ssh-keygen -t rsa -C “you@homestead”（git bash）
vagrant up
方便调试还需要修改windows本地hosts文件 192.168.10.10 homestead.app

homestead box

composer install
npm install
bower install
gulp
php artisan serve
